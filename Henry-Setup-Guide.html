Getting started with Henry

Henry schedules your crew and emails everyone their week in one click. It runs entirely on this computer — no account to create, nothing in the cloud. This guide takes you from install to sending your first schedule in about 15 minutes.

1Install Henry

On a Mac

  1. Open the Henry-1.0.0.dmg file you were given.
  2. Drag the Henry icon onto the Applications folder.
  3. Open Applications and double-click Henry.
First time only: if the Mac says Henry “can’t be opened” or is from an “unidentified developer,” right-click the Henry icon → Open → Open. If it still won’t open, go to System Settings → Privacy & Security, scroll down, and click Open Anyway. You only do this once.

On Windows

  1. Double-click Henry Setup 1.0.0.exe.
  2. Henry installs and opens on its own — no options to pick.
  3. Afterwards, open it any time from the Start menu or the desktop shortcut.
First time only: if Windows shows a blue “Windows protected your PC” box, click More info → Run anyway. You only do this once.

2First launch

The first screen asks “Protect this computer?” — an optional password that locks Henry itself, handy on a shared office computer.

  • Type a password twice and click Set password, or
  • Click Skip — you can always add one later in Settings → App password.

You then land on the Dashboard, which shows a short setup checklist. The next steps walk through it.

3Name your company

Go to Settings → Company (or use the Dashboard checklist). Type your company name and click Save name. It shows in the sidebar and signs the bottom of every schedule email, so set it before you publish anything.

4Connect your email

Henry sends schedules from your own email account. Set this up once in Settings → Email.

Gmail / Google Workspace (most common)

Gmail won’t accept your normal password from an app, so you create a one-time App Password:

  1. In a browser, sign in at myaccount.google.com with the Gmail account Henry should send from.
  2. Open Security and make sure 2-Step Verification is ON (App Passwords require it).
  3. Search the page for “App passwords,” open it, name one Henry, and click Create.
  4. Google shows a 16-character password once — copy it now.
  5. In Henry → Settings → Email, choose Gmail / Google Workspace and fill in:
From nameYour company name (what recipients see)
From addressThe Gmail address
UsernameThe same Gmail address
Gmail app passwordThe 16 characters (spaces don’t matter)

Click Save, then Send test email. Check that address’s inbox (and spam). If it arrives, you’re done.

Microsoft 365 / Outlook

Pick the Microsoft 365 / Outlook option and use the mailbox address and its normal password. If sign-in is blocked, your Microsoft admin needs to allow SMTP AUTH for that mailbox.

Any other provider

Pick Custom SMTP and enter the server, port, and security from your email provider (commonly port 587, TLS).

5Add your customers & employees

Two ways — do whichever is easier:

Type them in

  • Customers page → Add customer: name, address, opening hours, and an optional “email this customer” toggle.
  • Employees page → Add employee: name and, importantly, their email (that’s where their schedule is sent). You can also mark someone On call or record time off.

Or import a spreadsheet (faster for a full roster)

Open Import and drop in an Excel file (.xlsx) with two tabs named exactly CUSTOMER DATA and EMPLOYEE DATA, each with a header row on top. Henry shows a Review screen to confirm the columns before saving — nothing is saved until you click confirm. These columns auto-match:

CUSTOMER DATAEMPLOYEE DATA
Customer name requiredFull name required
Contact name, Phone, Address, City, ZIPEmail for delivery
Start time, End time (opening hours)EID, Position, E-Rating, Phone, City, State

Re-importing the same file later is safe — rows are matched by name and updated, never duplicated. So you can fix data in Excel and import again any time.

6Build a schedule

Open Schedule. Each row is a customer site; you place people into Morning and Afternoon shifts. Use the Day / Week toggle to switch views.

  • Assign: click a cell and pick a person, or drag someone between cells.
  • Notes: click an assignment to add a note (e.g. “bring the tall ladder”) — it goes into that person’s email.
  • Copy week: copy last week’s plan onto this week as a starting point.
  • Henry gently warns about time off, closed sites, and double-booking.

7Publish the week

Click Publish week in the Schedule toolbar.

  1. Optionally add a message at the top (goes into every email).
  2. Optionally pick who is on call — the backup person, who must not be scheduled to work that week.
  3. Click Preview the email to see exactly what a person will receive.
  4. Click Send to everyone. Each active employee gets one email with their own week plus the whole team’s plan.

If you edit a week you already sent, a reminder banner appears until you re-publish. Re-publishing only emails people whose week actually changed. Publish history lists every send, and you can resend any failed email from there.

8Activate your license

Each computer is activated by us — there’s no key to type. Open Settings → License, copy this computer’s license ID, and send it to us. We activate it on our side; you click Check again now and you’re set. Working offline is fine — Henry re-checks about once a day. Your data is never deleted.

9Back up your data

Everything lives in one folder on this computer (there is no cloud copy):

Mac~/Library/Application Support/henry
Windows%APPDATA%\henry

If this computer is backed up (Time Machine on Mac, File History / a backup tool on Windows), Henry is backed up with it. To move Henry to a new computer, install the app there and copy this folder across.

10If something goes wrong

ProblemFix
“Authentication failed” when sending (Gmail)You almost certainly used your normal password. Create an App Password (step 4) and paste that instead.
You changed your Gmail passwordThat revokes app passwords. Create a new one and update Settings → Email.
Test email never arrivesCheck spam; confirm the From address and Username are identical.
Import says a sheet is missingThe tabs must be named exactly CUSTOMER DATA and EMPLOYEE DATA.
Someone gets no emailThey have no email address on their Employees record, or they’re archived. Publish history shows the reason.
App won’t open after downloadMac: right-click → Open. Windows: More info → Run anyway.
